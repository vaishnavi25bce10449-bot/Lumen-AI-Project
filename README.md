## Project Overview
LUMEN is a specialized Command Line Interface (CLI) tool developed for the VITyarthi BYOP component. It utilizes **Unsupervised Machine Learning** to identify security threats and system irregularities within server log files.

By employing the **Isolation Forest** algorithm, LUMEN analyzes log data patterns (Severity, Message Length, and Complexity) to isolate outliers that could indicate system crashes or unauthorized access attempts.
